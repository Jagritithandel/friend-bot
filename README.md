# 🤖 Basic Chatbot (Friend Bot)

This is a simple rule-based chatbot built using Python.
The chatbot can respond to basic greetings and conversations using predefined responses.

---

## 🚀 Features

* Responds to greetings like "hi", "hello", "hey"
* Can answer simple questions like "how are you"
* Gives random responses for variety
* Handles unknown inputs with default replies
* Simple and beginner-friendly project

---

## 🛠️ Technologies Used

* Python
* Random Module

---

## 📂 Project Structure

```
basicbot.py   # Main chatbot file
```

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the file:

```
python basicbot.py
```

---

## 💬 Example Conversation

```
You: hi
Chatbot: hello!

You: how are you
Chatbot: I am doing great

You: bye
Chatbot: Goodbye! Have a great day!
```

---

## 🧠 How It Works

* The chatbot uses a dictionary of predefined responses
* It checks user input for keywords
* If a match is found, it returns a random response
* If no match is found, it returns a default response
